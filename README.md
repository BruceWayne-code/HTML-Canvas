### 📚 Topics Covered

This repository is organized into three main sections: **Basic**, **Intermediate**, and **Advanced**.

#### Basic
- **Drawing Shapes**: Lines, rectangles, circles, and polygons.
- **Paths & Strokes**: Using `beginPath()`, `lineTo()`, `arc()`, etc.
- **Colors & Styles**: `fillStyle`, `strokeStyle`, gradients, and patterns.
- **Text & Images**: Drawing text and manipulating images on the canvas.

#### Intermediate
- **Animations**: `requestAnimationFrame` and creating smooth animations.
- **Interactivity**: Handling mouse and keyboard events.
- **Transformations**: `translate`, `rotate`, `scale`, and `save`/`restore`.
- **Pixel Manipulation**: `getImageData()` and `putImageData()` for image effects.

#### Advanced
- **Game Development**: Building a simple game like Snake or Pong.
- **Data Visualization**: Creating bar charts, pie charts, and more.
- **Web Workers**: Offloading heavy computations to avoid blocking the main thread.
- **Performance Optimization**: Tips and tricks for high-performance canvas applications.
